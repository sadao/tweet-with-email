TEM へようこそ

Twitterにメールでつぶやけるサービスです。

Twitterアカウント毎に個別のメールアドレスを割り当てるので、それを利用してメールでつぶやくことができます。

ぜひ ログイン して使ってみてください！

このサイトの対象者

PCからインターネットができるが、携帯(PHS)からはインターネットに繋がない方。メールでつぶやきたい方。なぜならば、携帯(PHS)からサイトを見るのは従量制でお金が掛かるけど、メールはし放題だから！。つまり、Willcom Honey Bee 1 を従量制で契約しており、メールはし放題なのでメールでつぶやきたいのです。以前はTwitter公式の該当機能があったけど、今動かなくないですか？

利用条件

  • Twitterアカウントを作成する
  • TwitterからCunsumer Key と Consumer secret を取得する
  • TwitterからAccess Token と Access Token Secret を取得する
